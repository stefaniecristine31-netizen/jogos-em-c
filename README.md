# 🎮 Jogos em C

Repositório com projetos de jogos desenvolvidos em linguagem C, com foco no aprendizado de lógica de programação, algoritmos e resolução de problemas.

---

## 📌 Sobre

Este repositório reúne jogos criados durante meus estudos em programação, utilizando a linguagem C para praticar estruturas condicionais, loops, funções e desenvolvimento da lógica computacional.
